# 渲染与优化

css Render layer层，对于Render layer渲染会跳过Layout，paint，直接通知GPU对layer做变换 触发条件：

  • 根元素
  • 3dtransform属性或者透视
  • video元素
  • canvas和webgl
  • 使用animation,transition实现 opacity，transform的动画
  • fitters
  • clip
  • 有一个Z-index坐标比自己小的兄弟节点，该节点是一个合成层

css reLayout 重排

触发条件：

  • 页面的初始化
  • 盒子的变化
  • 添加和删除可见DOM
  • 元素位置，内容，或者尺寸变化
  • 浏览器窗口尺寸变化
  • 读取CSS属性

js读取属性引起重排的值

clientHeight,clientLeft,clientTop,clientWidth, focus(),getBoundingClientRect(),getClient
