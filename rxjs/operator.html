    # 操作符 operator

    pipe 用来处理操作符的，依次把流交给每个操作符

    map 和数组的 map 方法会产生新的数组类似，它会产生新的 Observabl

    import {from} from 'rxjs'
import {map} from 'rxjs/operators'
let doe$ = from([1,2,3])

doe$.pipe(
  map( x => x+1 )
).subscribe( x => console.log(x)) // 2 3 4

    mapTo 把每次的流都映射为一个固定的值

    import {from} from 'rxjs'
import {map, mapTo} from 'rxjs/operators'
let doe$ = from([1,2,3])

doe$.pipe(
  mapTo( 8 )
).subscribe( x => console.log(x))// 8 8 8

    filter 过滤和数组的filter类似

    import {from} from 'rxjs'
import {filter} from 'rxjs/operators'
let doe$ = from([1,2,3])

doe$.pipe(
  filter( x => x >  1)
).subscribe( x => console.log(x)) // 2 3

    scan 会把流的值循环求和 count是初始值，如果没指定则以第一个值为准，

    import {from} from 'rxjs'
import {filter, scan} from 'rxjs/operators'
let doe$ = from([1,2,3])
let count = 5;
doe$.pipe(
  scan( (x,y) => x + y, count)
).subscribe( x => console.log(x)) // 6 8 11
