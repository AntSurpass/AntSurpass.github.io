# css 定位

# float

用于更改页面自然流中元素的位置，并将其推送到其容器的左侧或右侧，使文本和内联元素环绕它，元素仍然存在与文档流中

  • none
  • left
  • right

当元素浮动时，除了下面几个，其他的display都为block

float前 float后
inline-table table
inline-flex flex
inline-grid grid

浮动元素注意的地方：

  1. 绝对定位元素不能浮动
  2. 浮动元素会保留在文档流中，所以会影响其他元素定位、
  3. 内联元素浮动时，会变为块级，即使设置了display为inlie-block等值，也会变为block
  4. 不会出现边距重叠
  5. 浮动元素的上边距边缘不会高于源文档中先前生成的任何块级别或浮动元素的边缘顶部
  6. 浮动元素必须尽可能高地放置在页面流中
  7. 左浮动元素的右边缘可能不在其右侧的任何右浮动元素的左边缘边缘的右侧
  8. 左浮动元素必须尽可能地放在左侧，右侧浮动元素尽可能向右

清除浮动的方法:

  • 在容器结束标签之前添加一个空标签并设置clear：both（缺点是会添加额外的标签，增大了文档体积）
  • 设置overflow为auto或hidden
  • 浮动元素父元素也浮动
  • :after 伪元素(推荐)
 .f:after {
    content: "."; 
    visibility: hidden; 
    display: block; 
    height: 0; 
    clear: both;
}

# position

用于设置页面上元素的位置,默认情况下所有元素都是静态元素（position：static）

除了静态元素，position还有下面几个值：

  • relative
  • absolute
  • fixed
  • sticky

上面的定位会创建定位元素，定位元素会创建一个定位上下文，可以用：top bottom left right 来指定偏移量（只对定位元素有效）

relative 相对定位元素，会为绝对定位创建定位上下文，本身也会创建定位上下文，可以使用偏移量从本身位置进行偏移，元素本身的位置任然在文档流中，不会影响其他元素布局
