    背景指元素内容，内边距，边框下面的区域

    背景可以简写为

    background: [ <bg-layer> , ]* <final-bg-layer>

/* where */
<bg-layer> = <bg-image> || <position> [ / <bg-size> ]? || <repeat-style> || <attachment> || <box>{1,2}

/* and */
<final-bg-layer> = <bg-image> || <position> [ / <bg-size> ]? || <repeat-style> || <attachment> || <box>{1,2} || <'background-color'>

    分别对应backgroun-image,background-position,background-size,background-repeat,background-origin,background-clip,background-attachment, 上面的 <position> [ / <bg-size> ]? 中的 / 是为了区分position和size

    背景属性及初始值

    • background-color: transparent
    • background-image: none
    • background-position 0% 0%
    • background-repeat: repeat
    • background-origin: padding-box
    • background-clip: border-box
    • background-attachment: scroll
    • background-size: auto auto
    • background-blend-mode: normal

    注意

    如果使用background简写并且仅指定一些背景属性而不是所有背景属性，则其他背景属性将在速记属性中设置为其初始值

    background: fixed

    相当于








     



    .element {
    background-color：transparent;    / *初始值* / 
    background-position: 0% 0%;       /*初始值*/ 
    background-size：auto auto;       / *初始值* / 
    background-repeat：repeat; / *初始值* / 
    background-clip：border-box;      / *初始值* / 
    background-origin：padding-box;   / *初始值* / 
    background-attachment：fixed;     / *来自简写属性的声明值* / 
    background-image：none;           / *初始值* / 
}

    注意

    如果存在多个背景，background-color只会作用于最后一个背景

    注意

    <box> 可以设置为padding-box，content-box，和border-box,只有一个值时，将会同时设置background-origin和background-clip，如果存在两个值，前者设置background-origin，后者设置background-clip

          background:  url(something.png) deepPink center center padding-box;

    相当于

        background-color: deepPink;
    background-image: url(1.jpg);
    background-repeat: repeat; /* initial */
    background-attachment: scroll; /* initial */ 
    background-position: center center;
    background-clip: padding-box; // 只设置一个box，所以clip和origin都是统一值
    background-origin: padding-box;
    background-size: auto auto;

    IE9以上可以设置多背景

        background: url(a.png) top left no-repeat,
               url(b.png) center / 100% 100% no-repeat,
               url(c.png) white;

    相当于

        background-image:      url(a.png),  url(b.png),          url(c.png);
    background-position:   top left,    center,              top left;
    background-repeat:     no-repeat,   no-repeat no-repeat, repeat;
    background-clip:       border-box,  border-box,          border-box;
    background-origin:     padding-box, padding-box,         padding-box;
    background-size:       auto auto,   100% 100%,           auto auto;
    background-attachment: scroll,      scroll,              scroll;
    background-color:      white;

    # background-repeat

    设置元素背景图像如何重复，有下面几个值

    • repeat 水平和竖直都重复（默认值）
    • repeat-x 水平方向重复
    • repeat-y 垂直方向重复
    • no-repeat 不重复
    • round 图像向两端对齐重复以填满容器，多余空间通过图像拉伸来填充 ie9+
    • space 图像向两端对齐重复以填满容器，多余空间用空白代替 ie9+
      background-repeat: no-repeat;
