    背景指元素内容，内边距，边框下面的区域

    背景可以简写为

    background: [ <bg-layer> , ]* <final-bg-layer>

/* where */
<bg-layer> = <bg-image> || <position> [ / <bg-size> ]? || <repeat-style> || <attachment> || <box>{1,2}

/* and */
<final-bg-layer> = <bg-image> || <position> [ / <bg-size> ]? || <repeat-style> || <attachment> || <box>{1,2} || <'background-color'>

    分别对应backgroun-image,background-position,background-size,background-repeat,background-origin,background-clip,background-attachment, 上面的 <position> [ / <bg-size> ]? 中的 / 是为了区分position和size

    背景属性及初始值

    • background-color: transparent
    • background-image: none
    • background-position 0% 0%
    • background-repeat: repeat
    • background-origin: padding-box
    • background-clip: border-box
    • background-attachment: scroll
    • background-size: auto auto
    • background-blend-mode: normal

    注意

    如果使用background简写并且仅指定一些背景属性而不是所有背景属性，则其他背景属性将在速记属性中设置为其初始值

    background: fixed

    相当于








     



    .element {
    background-color：transparent;    / *初始值* / 
    background-position: 0% 0%;       /*初始值*/ 
    background-size：auto auto;       / *初始值* / 
    background-repeat：repeat; / *初始值* / 
    background-clip：border-box;      / *初始值* / 
    background-origin：padding-box;   / *初始值* / 
    background-attachment：fixed;     / *来自简写属性的声明值* / 
    background-image：none;           / *初始值* / 
}

    注意

    如果存在多个背景，background-color只会作用于最后一个背景

    注意

    <box> 可以设置为padding-box，content-box，和border-box,只有一个值时，将会同时设置background-origin和background-clip，如果存在两个值，前者设置background-origin，后者设置background-clip

          background:  url(something.png) deepPink center center padding-box;

    相当于

        background-color: deepPink;
    background-image: url(1.jpg);
    background-repeat: repeat; /* initial */
    background-attachment: scroll; /* initial */ 
    background-position: center center;
    background-clip: padding-box; // 只设置一个box，所以clip和origin都是统一值
    background-origin: padding-box;
    background-size: auto auto;

    IE9以上可以设置多背景

        background: url(a.png) top left no-repeat,
               url(b.png) center / 100% 100% no-repeat,
               url(c.png) white;

    相当于

        background-image:      url(a.png),  url(b.png),          url(c.png);
    background-position:   top left,    center,              top left;
    background-repeat:     no-repeat,   no-repeat no-repeat, repeat;
    background-clip:       border-box,  border-box,          border-box;
    background-origin:     padding-box, padding-box,         padding-box;
    background-size:       auto auto,   100% 100%,           auto auto;
    background-attachment: scroll,      scroll,              scroll;
    background-color:      white;

    # background-color

    设置元素背景颜色，会在所有元背景图片下面（当元素背景图片可能无法加载时，可以颜色来做备用）

      background-color: #223322;
  backgrouns-color: hasl(20%,30%,40%,0.2);

    # background-image

    设置元素背景图像，默认情况在竖直和水平都是重复,在背景颜色上绘制，在border下方绘制,用逗号分割来指定多背景

      background-image: url('../image/1.png');
  background-image:url('../image/1.png'), linear-gradients(red,blue);

    # background-position

    指定背景图像在背景中的初始位置，指定的数据类型为<position>(关于<position>类型请看这里) 可以用关键字(top,left,center,right,bottom),百分比和长度来指定偏移值, 可以拆分为background-position-x,background-position-y

      background-position: 20px 20px;
  background-position: left center;

    相当于

      background-position-x: 20px;
  background-position-y: 20px;

  background-position-x: left;
  background-position-y: center;

    注意

    如果指定一个值，默认第二个值是center, 当指定3-4个值时ie9+，必须以关键字开头，后跟 % 或 长度单位的偏移值，如未指定偏移值则为0


     

       background-position:  20px bottom 20px; // error 无效的语法
   background-position: right bottom 20px; // 宾果

    相当于右下角向左移动0px和向上偏移20px

      background-position: right 0 bottom 20px;  // 如上，偏移值未设，值为0

    # background-repeat

    设置元素背景图像如何重复，有下面几个值

    • repeat 水平和竖直都重复（默认值）
    • repeat-x 水平方向重复
    • repeat-y 垂直方向重复
    • no-repeat 不重复
    • round 图像向两端对齐重复以填满容器，多余空间通过图像拉伸来填充 ie9+
    • space 图像向两端对齐重复以填满容器，多余空间用空白代替 ie9+
      background-repeat: no-repeat;

    # background-size

    指定背景图像的大小

     <bg-size> = [ <length> | <percentage> | auto ]{1,2} | cover | contain

    第一值为图像的宽度，第二个指图像的高度，可以使用长度，百分比，关键字来指定背景如何展示，可以用逗号来指定多背景

      background-size: 20px 2em;
  background-size: 20% 20%；
  background-size: cover;

    注意

    如果只指定一个值，默认第二个值是auto,无论是长度还是百分比都不能为负数

    关键字contain, 保持图像的宽高比到最大尺寸，使其宽度和高度都可以适应背景

    # background-origin

    指定背景图像定位原点位置(左上角) 关键字有 padding-box,border-box,content-box

    • padding-box: 上图（盒子模型）的padding左上角定位
    • border-box： border左上角
    • content-box: 内容区域左上角

    注意

    如何设置了background-attachment: fixed, 这该属性将无效

    # background-clip

    参照上面盒子模型图，指定背景绘制的区域，，有以下值，

    • padding-box 背景延伸到padding，不会在border下方绘制
    • border-box 延伸到border
    • content-box 只会绘制在内容区域
    • text 背景被裁剪成文字的 需要加上-webkit-前缀ie

    # background-attachment

    决定背景是在视口中固定的还是随包含它的区块滚动的。

    • fixed 相对于视口固定，不会随着元素内容滚动
    • local 相对元素内容固定，如果这个元素的内容是滚动的，背景将会一起滚动
    • scroll 相对与元素本身固定，不会随着元素内容滚动

    注意

    移动端兼容不好，安卓和open不支持
